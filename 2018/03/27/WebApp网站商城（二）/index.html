###如何在linux安装nodejs

  1. wget （http链接）
  2. xz -d (.xz文件) / tar -xzvf （.gz文件）
  3. tar -xvf (.tar文件)
  4. ln -s

如果出现xz not found command的错误：

  1. 到官网下载xz包：
    http://tukaani.org/xz/

  2. 解压安装包
    tar -jxvf （工具包名称）

  3. ./configure –prefix=/opt/gnu/xz

    ./configure的作用 是检测系统配置，生成makefile文件，以便你可以用make和make install来编译和安装程序。执行./configure的话要到你解压好的目录中去（cd 程序文件夹名称），也就是要在xz-5.0.4目录下使用这个命令

    ./configure是在源代码安装过程使用的，像xz-5.0.4.tar就是源代码包，但并不是所有的tar包都是源代码的包，可以先用ls看看有没有configure这个文件（xz-5.0.4目录下就有configure），没有的话就是二进制的包，如果是二进制的包，解压后直接就能使用，不需要使用./configure,因为它根本就没有configure文件。–prefix是个常用选项，–prefix=/opt/gnu/xz表示xz解压工具被安装到/opt/gnu/xz目录下

  4. make

  5. 安装xz
    sudo make install

  6. 创建软连接：
    ln -s /opt/gnu/xz/bin/xz /bin/xz

###如何在linux安装和配置MongoDB
wget http://downloads.mongodb.org/linux/mongodb-linux-x86_64-amazon-3.6.0.tgz （传送文件：scp 本地文件 远程服务器地址）
tar -zxvf mongodb-linux-x86_64-amazon-3.6.0.tgz

mkdir mongodb
mv mongodb-linux-x86_64-amazon-3.6.0 mongodb
cd mongodb
mkdir data
mkdir logs
cd logs/
touch mongo.log
cd ..
mkdir etc
cd etc/
vi mongo.conf

dbpath=/root/mongodb/data
logpath=/root/mongodb/logs/mongo.log
logappend=true
journal=true
quiet=true
port=27017
bind_ip = 127.0.0.1,XX.XX.XX.XX(远程连接的时候要加入服务器ip)
建立软连接：
ln -s /root/mongodb/mongodb-linux-x86_64-amazon-3.6.0/bin/mongod /usr/bin/mongod

在mongodb中创建用户

首先创建管理员
use admin
db.createUser({user:”root”,pwd:”123456”,roles:[{role:”root”,db:”admin”}]})
db.auth（“用户”，“密码”）

mongodb的基本用法

显示数据库：show dbs
创建数据库：use demo
删除数据库：db.dropDatabase()
删除集合：db.users.drop()
显示集合：show collections
创建文档：db.createCollection(“user”)
插入文档：db.users.insert({name:”kimmy”})
查找文档：
db.user.find()
db.user.findOne()
db.user.find({username:jack})
db.user.find({userAge:{$gt:40}})
db.user.find({userAge:{$gte:40}})
更改文档：db.user.update({username:”jack”},{$set:{userAge:30}})
删除文档：db.user.remove({userId:1})
数据的导入：mongoimport -d (数据库名称) -c （集合名称） –file （文件地址）

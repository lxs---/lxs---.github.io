2018-03-24
  • 在linux安装nodejs
  1. wget https://nodejs.org/dist/v8.11.1/node-v8.11.1.tar.gz
  2. tar -xvf node-v8.11.1.tar.gz
  3. ln -s (创建软链接)
  • 更新nodejs的方法
  1. 安装n模块：npm install -g n
  2. 升级node.js到最新稳定版：n stable
  3. 升级某个版本：n v0.12.2
  • nodejs调试方式
  1. webstorm：run -> edit configurations
  2. pm2
    pm2 start
    pm2 stop
    pm2 log
    pm2 list
  • 在linux安装MongoDB
  1. wget http://downloads.mongodb.org/linux/mongodb-linux-x86_64-amazon-3.6.0.tgz
    （或者：scp 本地文件 远程服务器地址）
  2. tar -zxvf mongodb-linux-x86_64-amazon-3.6.0.tgz

  • 在mongodb中创建用户
    首先要先创建管理员
    use admin
    db.createUser({user:”root”,pwd:”123456”,roles:[{role:”root”,db:”admin”}]})
    db.auth（“用户”，“密码”）

  • mongodb的基本用法
    显示数据库：show dbs
    创建数据库：use demo
    删除数据库：db.dropDatabase()
    删除集合：db.users.drop()
    显示集合：show collections
    删除集合：db.users.drop()
    创建文档：db.createCollection(“user”)
    插入文档：db.users.insert({name:”kimmy”})
    查找文档：
    db.user.find()
    db.user.findOne()
    db.user.find({username:jack})
    db.user.find({userAge:{$gt:40}})
    db.user.find({userAge:{$gte:40}})
    更改文档：db.user.update({username:”jack”},{$set:{userAge:30}})
    删除文档：db.user.remove({userId:1})
    数据的导入：mongoimport -d [数据库名称] -c [集合名称] –file [要倒入的文件地址]
